## Tech Stacks

- Next.js on Vercel
- Firebase authentication, Firestore
- TypesScript

## Browser Support

It has been checked to work with the following browsers.

- Google Chrome
- Firefox
- Microsoft Edge
- Safari

```
NEXT_PUBLIC_FIREBASE_APP_ID
```
